# Clucknest

## Project setup
Install Vue Cli globally

```
npm install -g @vue/cli
```

cd into project directory and run:
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration
